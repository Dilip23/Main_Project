            For each person with id between [L, R] :

            Sort Q in increasing order of the beauty value of the persons
            Flag = 0;
            For each person's id in Q : 
                Flag = 1 - Flag

        DiffSum (Z[1...n]) :
            Sum = 0
            For i = 2 to n :
                u = Z[i]
                v = Z[i - 1]
                Sum = Sum + |P[u] - P[v]|
            Return Sum
