<p class=" mathjax-support">Let's define the Maximum Repeating Block (MRB) of a string S, as the largest integer K, for which there exists another string Y, which when repeated K times, produces the string S.</p>
<ul class=" mathjax-support">
<li class=" mathjax-support">For example, the MRB of "hellohellohello" is 3, because there exists a string Y="hello" which when repeated 3 times produces the string "hellohellohello".</li>
<li class=" mathjax-support">Notice that K=1 also works, when the string Y="hellohellohello", but as K=3 is the maximum value, 3 is the MRB.</li>
</ul>
